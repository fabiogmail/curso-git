Java Runtime Environment
O "Java(TM) 2 Runtime Environment", ou JRE, contém a máquina virtual Java, as classes de bibliotecas e o inicializador de aplicações Java que são necessários para executar programas escritos na linguagem de programação Java.
O plug-in aqui disponível permite que o Internet Explorer suporte aplicações que utilizem a plataforma Java 2, que é o caso do CDRView Cliente.
