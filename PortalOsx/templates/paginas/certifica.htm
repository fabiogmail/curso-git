Introdução
Um applet é um aplicativo desenvolvido utilizando-se a linguagem de programação Java. Ele é executado automaticamente quando baixado pela rede (WEB). Por natureza, um applet não tem permissão para acessar recursos locais tais como os discos rígidos. Dessa forma, ele é incapaz de apagar dados de seu HD. Contudo, essa restrição também não possibilita que um applet escreva arquivos no HD.

Porém, existe uma maneira de garantir a um applet permissões de acesso a recursos locais. Isso é feito gerando-se uma assinatura digital e agregando-a ao arquivo contendo o applet. Diz-se, então, que o applet é digitalmente assinado.

Certificação
Certificação é o processo de instalação de um certificado que permitirá a verificação e validação de um applet digitalmente assinado. Caso o applet seja validado, ele poderá utilizar os recursos locais da máquina na qual está sendo executado.

O CDRView Cliente
O CDRView Cliente é um applet e, como já citado anteriormente, não é capaz de acessar os recursos locais. Isto lhe impossibilita de salvar os relatórios gerados no disco rígido local. Porém, o CDRView Cliente é um applet digitalmente assinado. Uma vez que o seu respectivo certificado tiver sido instalado, ele poderá utilizar os recursos locais, podendo realizar a exportação de relatórios gerados para arquivos.

É importante ressaltar que o acesso pelo CDRView Cliente aos seus discos rígidos não causará nenhum dano a sua estrutura de arquivos.

Utilização
